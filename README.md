# 음식 검색 APP - 3인 프로젝트 
## 목적 ##
+ 무엇을 먹을지에 대한 고민 해결
+ 내 주변 음식점 검색
+ 빠르고 편리한 설계
------------
# 개발 환경 
> Language
+ JAVA
+ php
+ 닷홈 phpMyAdmin DB
> Platform, API
+ Android Studio v4.1.1
+ Visual Studio Code v1.54.3
+ Adobe PhotoShop 2021
+ Google Map API
+ Google Places API
+ Google Maps SDK for Android
------------
